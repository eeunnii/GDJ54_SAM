갤러리 커뮤니티

회원
  - 가입/탈퇴/로그인/로그아웃
  - 아이디찾기/비밀번호찾기
  - 정보변경/비밀번호변경
  - 네이버로그인/카카오로그인
자유 게시판
  - 다단 계층 게시판
  - 로그인한 유저만 작성 가능
  - 작성자만 삭제 가능
  - 작성자만 내용 수정 가능
갤러리 게시판
  - 게시글에 사진을 올릴 수 있는 게시판
  - 댓글 작성 가능
  - 좋아요 기능
  - 글 작성 시 포인트 지급
업로드/다운로드 게시판
  - 로그인한 유저만 작성 가능
  - 작성자는 파일 업로드 가능
  - 작성자만 삭제 가능
  - 작성자만 내용/첨부 수정 가능
  - 상세보기에서 다운로드 지원
  - 글 작성 시 포인트 지급
관리자
  - 부트스트랩
  - 아이디가 admin인 유저가 관리자(미리 DB에 저장해 둘 것)
  - 관리자로 로그인하면 관리자 페이지로 이동하는 링크 보임
  - 별도 관리자 페이지에서 작업
  - 회원관리/자유게시판관리/갤러리게시판관리/업로드/다운로드 게시판
